# 智慧校园学情与社团一体化管理小程序

基于微信小程序的智慧校园管理平台，整合社团运营与学情数据管理两大核心场景。

## 📋 项目简介

- **学生端**：社团报名、活动签到、成绩查询、考勤记录、综测加分
- **教师/管理员端**：社团管理、成员审批、活动发布、学情看板可视化、批量操作
- **技术方案**：微信小程序原生框架 + Trae 数据可视化

## 🚀 快速开始

### 环境准备

1. 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 注册微信小程序账号（可选，测试可使用测试号）

### 项目运行

1. 使用微信开发者工具打开项目根目录 `smart-campus`
2. 在 `app.js` 中配置 Trae 后端地址（如使用 Trae）
3. 点击「编译」按钮预览

## 📁 项目结构

```
smart-campus/
├── pages/                  # 页面目录
│   ├── index/             # 首页
│   ├── login/             # 登录
│   ├── clubs/             # 社团模块（列表/详情/报名/我的）
│   ├── activities/        # 活动模块（列表/详情）
│   ├── signin/            # 签到
│   ├── grades/            # 成绩查询
│   ├── attendance/        # 考勤记录
│   ├── awards/            # 综测加分
│   ├── messages/          # 消息中心（列表/详情）
│   ├── feedback/          # 意见反馈
│   ├── profile/           # 个人中心
│   └── admin/             # 管理端
├── utils/                 # 工具函数
│   ├── util.js           # 工具函数
│   ├── api.js            # API 接口
│   └── mockData.js       # 模拟数据
├── assets/                # 静态资源
├── app.js                 # 小程序入口
├── app.json               # 小程序配置
├── app.wxss               # 全局样式
└── project.config.json    # 项目配置
```

## 🎯 核心功能

### 学生端功能

| 功能模块 | 描述 |
|---------|------|
| 社团列表 | 分类浏览、搜索社团 |
| 社团详情 | 介绍、成员、活动记录 |
| 社团报名 | 在线提交申请、查看审批 |
| 活动签到 | 扫码/点击签到、签到记录 |
| 成绩查询 | 按学期查询成绩、平均分统计 |
| 考勤记录 | 考勤状态、考勤率统计 |
| 综测加分 | 加分项目、审核状态 |
| 消息中心 | 社团通知、学习提醒、审批结果 |
| 意见反馈 | 提交反馈、查看回复 |

### 教师/管理员功能

| 功能模块 | 描述 |
|---------|------|
| 社团管理 | 新增、编辑、下架社团 |
| 成员审批 | 审核入团申请 |
| 活动发布 | 创建活动、生成签到二维码 |
| 签到管理 | 查看签到记录、导出 |
| 学情看板 | 考勤率柱状图、成绩分布饼图、综测排名折线图 |
| 批量操作 | 批量导入、批量导出、异常预警 |

## 🎨 设计风格

- 蓝白色调，简约校园风格
- 卡片式布局，清晰的视觉层次
- 响应式设计，适配各机型
- 流畅的交互体验

## 📊 数据模型

项目包含以下核心数据表（见 `utils/mockData.js`）：

1. users - 用户表
2. clubs - 社团表
3. club_members - 社团成员表
4. activities - 活动表
5. signin_records - 签到表
6. grades - 成绩表
7. attendance - 考勤表
8. awards - 综测表

## 📝 使用说明

### 学生端

1. 点击「首页」的快捷入口进入对应功能
2. 点击「社团」标签浏览和报名社团
3. 点击「学情」标签查看成绩、考勤和综测
4. 点击「消息」查看通知
5. 点击「我的」查看个人信息和退出

### 管理端

1. 在登录页选择「教师」身份
2. 在首页快捷入口进入管理功能
3. 在学情看板查看班级数据统计
4. 使用批量导入导出功能处理数据

## 🔧 配置说明

在 `app.js` 中配置：
- `apiBaseUrl` - 后端 API 地址
- 全局用户登录状态
- Trae 数据可视化配置

## 🚀 部署上线

1. 在微信公众平台注册小程序账号
2. 配置服务器域名（如需要后端）
3. 在微信开发者工具中上传代码
4. 提交审核
5. 审核通过后发布上线
